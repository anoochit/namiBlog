# Mastering Agent Skills

Agent Skills คือกุญแจสำคัญที่ทำให้ Nami เก่งขึ้นเรื่อยๆ โดยการเพิ่มความเชี่ยวชาญเฉพาะทางลงไป นี่คือสรุปสั้นๆ สำหรับการสร้างและจัดการทักษะใหม่ค่ะ

## ทำไมต้องสร้าง Skill?
เมื่อเราต้องการให้ Nami ทำงานเฉพาะทางได้อย่างแม่นยำ การสร้าง Skill คือคำตอบที่ทรงพลังที่สุด ช่วยลดการอธิบายซ้ำๆ และทำให้ Workflow มีประสิทธิภาพสูงขึ้น

## วิธีสร้าง Skill
คุณสามารถสร้างได้ 2 วิธี:

### 1. วิธีแมนนวล (Manual)
1. สร้างโฟลเดอร์ใน `workspace/.skills/`
2. เขียนไฟล์ `SKILL.md` ที่ประกอบด้วย:
   - **Name:** ชื่อของ Skill
   - **Description:** สรุปหน้าที่ของ Skill
   - **Instructions:** วิธีการและเงื่อนไขการทำงาน
   - **Resources:** ไฟล์หรือเทมเพลตที่ต้องใช้

### 2. ใช้เครื่องมือ (Skill Creator)
เรียกใช้งานผ่านคำสั่งของ Nami โดยระบุชื่อและรายละเอียด เครื่องมือจะจัดโครงสร้างให้คุณโดยอัตโนมัติ

## ข้อแนะนำ
- รักษาความกระชับ (Keep it concise).
- ตรวจสอบความปลอดภัยตามแนวทางของโครงการ.
- อย่าลืมทดสอบด้วยการลองเรียกใช้ เพื่อดูว่า Nami ทำงานได้ตามคาดหวังหรือไม่.
