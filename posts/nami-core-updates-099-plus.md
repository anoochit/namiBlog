# อัปเดต Nami Core: สรุปฟีเจอร์เด่นเวอร์ชัน 0.9.9 ขึ้นไป

สวัสดีครับชาว Nami Core ทุกท่าน! วันนี้เรามีความตื่นเต้นที่จะนำเสนอสรุปการอัปเดตครั้งสำคัญของ Nami Core ในเวอร์ชัน 0.9.9, 0.9.10 และ 0.9.11 ที่ผ่านมา ซึ่งเต็มไปด้วยฟีเจอร์ใหม่ การปรับปรุงประสิทธิภาพ และการแก้ไขข้อผิดพลาดที่สำคัญ เพื่อให้แพลตฟอร์มของคุณทำงานได้ดียิ่งขึ้น

เราได้มุ่งเน้นไปที่การขยายขีดความสามารถในการผสานรวม การปรับปรุงประสบการณ์ผู้ใช้ และการเพิ่มความแข็งแกร่งของสถาปัตยกรรมภายใน เพื่อตอบสนองความต้องการที่หลากหลายของนักพัฒนาและผู้ใช้งาน ไม่ว่าจะเป็นการเชื่อมต่อกับแพลตฟอร์มใหม่ๆ การรองรับ AI ระดับองค์กร หรือการปรับปรุง UI ให้ใช้งานง่ายขึ้น มาร่วมดูกันว่ามีอะไรใหม่บ้างในแต่ละเวอร์ชัน!

## [0.9.11] - 2026-05-13: การรวม LINE Bot และการปรับปรุงขั้นตอนการเริ่มต้นโปรเจกต์

เวอร์ชันนี้เป็นก้าวสำคัญในการขยายช่องทางการสื่อสารของ Nami Core ด้วยการเพิ่มการรองรับ LINE Messaging Platform อย่างเต็มรูปแบบ พร้อมทั้งปรับปรุงกระบวนการตั้งค่าโปรเจกต์ให้เป็นระเบียบยิ่งขึ้น

### ฟีเจอร์เด่น:
*   **การผสานรวม LINE Bot**: ตอนนี้ Nami Core สามารถทำงานเป็น LINE Bot ได้แล้ว! เราได้เพิ่มโหมดการทำงานใหม่สำหรับ LINE โดยเฉพาะ รวมถึงคำสั่ง `line` ใน CLI, การใช้งาน webhook server ด้วย Axum พร้อมการตรวจสอบลายเซ็น HMAC-SHA256 และการผสานรวมกับ LINE Messaging API เพื่อตอบกลับข้อความผู้ใช้
*   **จัดระเบียบการเริ่มต้นโปรเจกต์ใหม่**: คำสั่ง `init` ได้รับการปรับปรุงใหม่ให้เป็นขั้นตอน 4 ขั้นตอนที่ชัดเจน (LLM, Search, Bot, Observability) พร้อมส่วนหัวที่สวยงามและการจัดกลุ่มพร้อมท์ที่เข้าใจง่ายขึ้น
*   **การกำหนดค่า Observability**: เพิ่มการรองรับการกำหนดค่า URL ของ OpenTelemetry collector (`OTEL_COLLECTOR`) ระหว่างการเริ่มต้น
*   **เอกสารประกอบที่ปรับปรุงใหม่**: อัปเดต `README.md` ของโมดูล `modes` ด้วยคำอธิบายที่ครอบคลุมสำหรับโหมดการทำงานทั้งหมด รวมถึง LINE และคำสั่ง `run` โดยตรง

## [0.9.10] - 2026-05-12: รองรับ Vertex AI และการปรับปรุงด้านโครงสร้าง

เวอร์ชันนี้มุ่งเน้นไปที่การขยายขีดความสามารถด้าน AI ด้วยการรองรับ Google Cloud Vertex AI และการปรับปรุงโครงสร้างภายในเพื่อให้ Nami Core ทำงานได้อย่างมีประสิทธิภาพและยืดหยุ่นมากยิ่งขึ้น

### ฟีเจอร์เด่น:
*   **รองรับ Vertex AI**: เพิ่มการรองรับ Google Cloud Vertex AI อย่างเป็นทางการ ทำให้สามารถผสานรวมกับโมเดล Gemini ระดับองค์กรได้อย่างราบรื่นโดยใช้ Application Default Credentials (ADC)
*   **การกำหนดค่าตามภูมิภาค**: เพิ่มฟิลด์ `project_id` และ `location` ไปยัง `ModelConfig` และ `ReflectionConfig` เพื่อรองรับการใช้งาน LLM แบบภูมิภาค และข้อกำหนดด้าน residency ของ Google Cloud
*   **การตั้งค่า Vertex แบบโต้ตอบ**: คำสั่ง `init` ได้รับการอัปเดตให้สามารถกำหนดค่า Vertex AI แบบโต้ตอบได้ โดยจะตรวจจับและพร้อมท์สำหรับพารามิเตอร์ Google Cloud ที่จำเป็นโดยอัตโนมัติ
*   **ตรรกะการโหลดโมเดลที่ดีขึ้น**: ปรับปรุง `load_model` ให้ข้ามการตรวจสอบตัวแปรสภาพแวดล้อม API key เมื่อใช้ "vertex" provider ซึ่งจะไปพึ่งพา ADC ระดับระบบแทน
*   **ความเข้ากันได้ของ Tool Schema สำหรับ Vertex AI**: แก้ไขข้อกำหนด schema สำหรับเครื่องมือต่างๆ เพื่อให้เข้ากันได้กับ `OBJECT` parameters schema ของ Vertex AI

## [0.9.9] - 2026-05-11: ประสบการณ์ผู้ใช้ WebUI และการปรับปรุงการสตรีมข้อความ

เวอร์ชันนี้เน้นที่การปรับปรุงประสบการณ์ผู้ใช้ใน WebUI และการแก้ไขปัญหาที่เกี่ยวข้องกับการแสดงผลและการสตรีมข้อความ โดยเฉพาะอย่างยิ่งการรองรับภาษาไทย

### ฟีเจอร์เด่น:
*   **ตัวบ่งชี้สถานะเซิร์ฟเวอร์**: เพิ่มคอมโพเนนต์สถานะการเชื่อมต่อเซิร์ฟเวอร์แบบเรียลไทม์ในแถบด้านข้างของ WebUI เพื่อให้ข้อเสนอแนะทางสายตาเกี่ยวกับสถานะแบ็กเอนด์
*   **การนำทางประวัติพร้อมท์**: เปิดใช้งานการนำทางด้วยปุ่มลูกศร (ขึ้น/ลง) ในช่องป้อนข้อมูลของ WebUI เพื่อเรียกดูประวัติพร้อมท์ก่อนหน้า
*   **การแสดงผลการเรียกใช้ Tool**: เพิ่มคอมโพเนนต์ UI เฉพาะ (`ToolAccordion`) ใน WebUI เพื่อแสดงการเรียกใช้เครื่องมือและผลลัพธ์ในรูปแบบที่มีโครงสร้างภายในเธรดแชท
*   **การนำเสนอผลลัพธ์ Tool ที่ปรับปรุงใหม่**: ปรับปรุงคำแนะนำสำหรับ persona ของ Agent ให้สรุปผลลัพธ์ของเครื่องมือด้วยภาษาธรรมชาติที่กระชับ ห้ามแสดงผล JSON ดิบในแชท
*   **การแสดงผล Markdown ที่ปรับให้เหมาะสม**: ปรับปรุงการแสดงผล `ReactMarkdown` ใน WebUI ด้วยการปรับระยะห่างของย่อหน้า (`prose-p:m-0`) เพื่อให้เลย์เอาต์ข้อความดูสะอาดตาขึ้น
*   **ตรรกะการสตรีมที่ดีขึ้น**: ปรับปรุงตัวประมวลผลข้อความล่วงหน้าสำหรับการสตรีมใน `useChat.ts` เพื่อจัดการการเปลี่ยนผ่านอักขระภาษาไทย/ไม่ใช่ภาษาไทย และการจัดรูปแบบตัวทำเครื่องหมายรายการ ป้องกันการเชื่อมคำที่ไม่พึงประสงค์
*   **แก้ไขความกำกวมของ Task**: เปลี่ยนชื่อศัพท์เฉพาะของ Todo-tool (จาก "task" เป็น "todo item") เพื่อป้องกันความสับสนกับ State Manager tasks

## สรุป

การอัปเดตเหล่านี้แสดงให้เห็นถึงความมุ่งมั่นของเราในการพัฒนา Nami Core ให้เป็นแพลตฟอร์มที่ทรงพลัง ยืดหยุ่น และใช้งานง่ายยิ่งขึ้น การเพิ่มการผสานรวมกับ LINE และ Vertex AI จะช่วยเปิดโอกาสใหม่ๆ สำหรับนักพัฒนาในการสร้างแอปพลิเคชันที่ซับซับซ้อนและมีประสิทธิภาพ ส่วนการปรับปรุง UI และการแก้ไขข้อผิดพลาดต่างๆ จะช่วยให้ประสบการณ์การใช้งานโดยรวมราบรื่นและน่าพึงพอใจมากยิ่งขึ้น

เราขอแนะนำให้ผู้ใช้ทุกท่านอัปเดตเป็นเวอร์ชันล่าสุดเพื่อสัมผัสกับฟีเจอร์ใหม่ๆ และการปรับปรุงเหล่านี้ หากมีข้อเสนอแนะหรือพบปัญหาใดๆ อย่าลังเลที่จะแจ้งให้เราทราบผ่านช่องทางปกติ เรามุ่งมั่นที่จะพัฒนา Nami Core ให้ดียิ่งขึ้นไปพร้อมกับชุมชนของเรา! ขอบคุณที่ให้การสนับสนุนเสมอมาครับ!
